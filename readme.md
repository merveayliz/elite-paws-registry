# 🐾 The Paws Registry: Elite Squad

A modern, responsive, and dark-themed pet portfolio website built with clean HTML and CSS. This project focuses on high-quality typography effects, CSS layouts (Flexbox), and mobile-first responsive design.

## 🌟 Features
- **Neon-Infused Dark UI:** Inspired by terminal-style aesthetics.
- **Advanced CSS Typography:** Custom titles using `linear-gradient` text filling and `-webkit-text-stroke` effects.
- **Fully Responsive:** Adapts perfectly from 4K monitors down to mobile screens (2x2 grid layout on mobile).
- **Aspect-Ratio Optimized:** Vertical rectangular card designs for a cinematic feel.

## 🛠️ Tech Stack
- **HTML5** (Semantic structure)
- **CSS3** (Flexbox, Media Queries, Gradients, Webkit effects)
- **Google Fonts** (Fira Code for that technical vibe)
